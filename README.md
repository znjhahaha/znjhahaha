## 👨‍💻 关于我

- 🔭 **当前项目**：正方教务系统全平台生态（Web、Android、Cloud）
- 🌱 **正在钻研**：Next.js 15、Cloudflare D1、Kotlin 协程与高性能抢课算法
- ⚡ **核心理念**：用技术解决重复性劳动，让校园生活更高效
- 💬 **交流方向**：React、Android 原生开发、自动化脚本、Cloudflare Workers

## 🚀 核心开源项目

- 💻 **[zhengfangqk](https://github.com/znjhahaha/zhengfangqk)**  
  现代化的正方教务 Web 客户端。采用 Next.js + TypeScript + Tailwind CSS 构建，支持智能抢课、成绩查询与美化课表。

- 📱 **[zhengfang-apk](https://github.com/znjhahaha/zhengfang-apk)**  
  原生 Android 客户端。使用 Kotlin/Java 开发，支持后台服务监控、定时任务与一键抢课。

- ☁️ **[web-app](https://github.com/znjhahaha/web-app)**  
  基于 Cloudflare 生态的云端抢课分发系统。利用 Workers 与 D1 SQLite 实现分布式任务调度与管理后台。
